# KenEyeCue AI 協作規則

本檔供 GitHub Copilot、Claude Code、Codex、Cursor、ChatGPT Agent 與其他 AI 工具讀取。所有修改必須依照以下規則執行。

## 1. 核心定位

KenEyeCue 不是算命服務，也不是一般修眉服務。

KenEyeCue 是：

> 以眉眼訊號為核心的第一印象校準系統。

固定用語：

- 對外主語：眉眼訊號
- 專業分析語：眉眼結構
- 高端方案語言：視覺權威訊號
- 核心交付：第一印象校準

## 2. 嚴格禁止

不得新增或保留以下承諾式語言：

- 一定會
- 必然
- 保證
- 100%
- 改運成功
- 招財保證
- 開運眉
- 註定
- 命不好
- 凶、煞等恐嚇式判語

命理、紫微、八字、易經、塔羅、占星相關內容，只能作為象徵性語言與結構傾向參考，不得作絕對預測。

## 3. AI 修改原則

AI 可以做：

- 初稿
- 摘要
- 複審
- 文件整理
- UI 文案建議
- 測試清單
- PR 檢查

AI 不可以做：

- 替 Ken 做最終品牌定稿
- 自行新增未確認命盤資料
- 自行定義十四主星最終版本
- 自行承諾商業結果、感情結果、健康結果或投資結果

## 4. 每次修改前必須先判斷

修改任何內容前，先確認：

1. 這次修改是否符合 KenEyeCue 核心定位？
2. 是否有補腦或未確認資料？
3. 是否有把命理說成絕對預測？
4. 是否有把品牌拉回眉眼訊號、眉眼結構與第一印象校準？
5. 是否保留 Ken 最終判斷權？

## 5. 輸出格式

所有策略、品牌、內容、命理轉譯、行銷類回覆，預設使用：

```text
✅ 已知事實
💡 合理推論
🎯 策略建議
⚠️ 待確認
最終定案
```

## 6. 標準聲明

涉及命理象徵分析時，結尾必須保留：

> 以上為象徵性趨勢分析，用於調整方向，不構成對任何具體結果的保證。
